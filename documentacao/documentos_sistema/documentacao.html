1. Introdução

O Sistema de Garantias de Baterias foi desenvolvido para facilitar o gerenciamento de garantias de baterias coletadas durante o trabalho de motoristas. Ele permite o registro, consulta, edição e exclusão de lançamentos de garantias com informações detalhadas, como o modelo da bateria, código, rótulo, data de coleta e dados do cliente.

2. Tecnologias Utilizadas

  • Frontend: HTML, CSS
  • Backend: PHP
  • Banco de Dados: MySQL
  • Ferramentas de Desenvolvimento: XAMPP (para rodar o servidor Apache e MySQL)

3. Funcionalidades

  1. Tela de Login: Acesso ao sistema por meio de credenciais de usuário.
  2. Tela Principal: Exibição de uma lista de lançamentos de garantias com opções de editar, excluir e buscar registros.
  3. Lançamento de Garantias: Registro de novas garantias de baterias com informações como estado, cidade, cliente, modelo da bateria, código da bateria, rótulo e data de coleta.
  4. Consulta e Edição de Lançamentos: Permite consultar e editar registros de garantias existentes.
  5. Busca e Filtro: Filtragem de lançamentos por cliente, cidade e data de coleta.

4. Como Instalar

Siga os passos abaixo para configurar o sistema em seu ambiente local:

  • 1. Baixe ou clone o repositório do projeto.
  • 2. Configure o banco de dados: Acesse o phpMyAdmin ou use um cliente MySQL e crie um banco de dados chamado sistema_garantias. Importe o arquivo SQL para criar as tabelas.
  • 3. Configure o servidor web: Coloque os arquivos no diretório do servidor Apache (ex.: htdocs no XAMPP) e inicie o Apache e MySQL.
  • 4. Ajuste as configurações de conexão com o banco de dados: No arquivo conexao.php, edite as credenciais do banco de dados.
  • 5. Acesse o sistema: Abra o navegador e vá para http://localhost/sistema-garantias/login.html.

5. Como Usar

  1. Login: Na tela de login, insira as credenciais de usuário e clique em "Entrar".
  2. Tela Principal: Após o login, será exibida a lista de lançamentos de garantias. Você pode buscar, filtrar e editar registros existentes.
  3. Adicionar Lançamento: Clique para adicionar um novo lançamento de garantia. Preencha os campos necessários e clique em "Salvar".
  4. Editar ou Excluir Lançamento: Na lista de lançamentos, você pode editar ou excluir registros existentes.
